<H1>Keydrop api gateway</H1>

## TechStack

- Next.js
- Redis
- Mysql
- Docker

## Installation locally
```
create .env file in root directory and copy content from .env.example
```

```
$ docker-compose up
```

```bash
$ npm install -g @nestjs/cli
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
